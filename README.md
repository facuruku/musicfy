# Musicfy

App to enjoy your favorite music anywhere!

[Click here to try it!](https://facuruku.github.io/musicfy/#/)

## Features

- Listen to your music ONLINE or OFFLINE
- Installable PWA
- Responsive

## Tech Stack

**Client:** Vue, TailwindCSS

**Server:** Vite

**Database:** Firebase

**Testing:** Cypress

## Run Locally

Clone the project

```bash
  git clone https://github.com/facuruku/musicfy.git
```

Go to the project directory

```bash
  cd musicfy
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
